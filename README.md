# CrackAnalysis

裂隙自动识别程序V1.0、V2.0、V3.0

liexi_18、liexi_20以及liexi_24cos是我申请三个软件著作权的源码

liexi_24xx以及liexi_24cosxx分别是固定学习率及余弦退火学习率下训练模型的代码
